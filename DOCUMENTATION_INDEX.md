## üí° Key Concepts

### Server vs Client Components
- **Server Components**: Fetch data, access file system
- **Client Components**: Handle animations, user interactions
- **API Routes**: Bridge between server and client
‚Üí Read: [EDGE_RUNTIME_FIX.md](EDGE_RUNTIME_FIX.md)

### Edge Runtime
- Runs your code in 300+ global locations
- <10ms latency to users anywhere
- Optimized for serverless functions
‚Üí Read: [EDGE_RUNTIME.md](EDGE_RUNTIME.md)

### ISR (Incremental Static Regeneration)
- Poems list: Pre-rendered at build, revalidate every 1 hour
- Individual poems: Generated on-demand when first requested
- New poems automatically available after next revalidation
‚Üí Read: [README.md](README.md)

### Sveltia CMS
- Free forever Git-based CMS
- No database needed (uses GitHub)
- Edit poems without code
- All changes pushed to GitHub
‚Üí Read: [CMS_GUIDE.md](CMS_GUIDE.md)
